  • Параметры балкона
    • Площадь пола = 7.5
    • Высота поднятия пола = 0.1
    • Высота потолков = 2.65
    • Ширина боковой поверхности = 1.3
    • Оконный проем на кухне = 1.48 x 0.65
    • Балконный проем на кухне = 2.16 x 0.81
    • Оконный проем в зале = 1.48 x 1.6
    • Общая длина балкона = 5.77
    • Боковая поверхность = 2.65 x 1.3
    • Основная поверхность за вычетом оконных проемов = 5.77 x 2.65 - (1.48 x 0.65) - (2.16 x 0.81) - (1.48 x 1.6) = 10.21
    • Глубина оконных проемов = 0.15
    • Площадь пола и потолка = 2 x 7.5 = 15
    • Общая площадь стен = 2 x 2.65 x 1.3 + 10.21 = 17.1
    • Общая площадь поверхности = 2 x 2.65 x 1.3 + 2 x 7.5 + 10.21 = 32.1
